## 🔄 End-to-End Data Flow

```
Frontend (React) 
    ↓ (API calls via services/api.ts)
Backend (Rust/Axum)
    ↓ (Action dispatch via modules.rs)
Python Tools (50+ modules)
    ↓ (Processing & results)
Backend (Response formatting)
    ↓ (JSON response)
Frontend (UI update)
    ↓ (WebSocket for real-time)
User sees results
```
